# 00 — Source

Raw, unprocessed inputs. **Nothing here is Brand Truth yet.** This is the holding pen for everything we gather before distilling it: stakeholder and founder interviews, brand audits, competitive scans, existing-asset inventories, decks, transcripts, and reference links.

## What goes here
- Verbatim interview and workshop notes
- Brand audits and heuristic reviews
- Inventories of existing assets (sites, packaging, decks)
- External references and inspiration

## How to author
- One file per artifact. Use a descriptive kebab-case filename.
- Start every file with a header line noting **where it came from** and **the date** — e.g. `> Source: founder interview (J. Rivera) — 2026-06-28`.
- Keep it raw. Don't polish, edit, or pre-decide. Contradictions are fine and expected at this stage.

## What happens next
Truths are **distilled** from these raw inputs into `01-imprint`, `02-voice`, and `03-design`. Each distilled truth should cite its source (often a file in here) in its `source:` frontmatter field. This folder is **exempt** from the truth frontmatter contract.
